# SC Assets

This tree is the minimal Phase 7 primitive bootstrap layer.

- `families/*/probe.scd`:
  draft probe patches for governed exploration loops
- `families/*/final-nrt.scd`:
  NRT source files that return an Event with `score`, `duration`, `sample_rate`,
  `sample_format`, and `channel_count`
- `families/*/candidate-summary.md`:
  lightweight summary templates for archive and candidate review; `sustained-tonal-carrier` is filled as a reference example, other families remain templates until dogfood
- `patterns/*.scd`:
  narrow pattern scaffolds that stay inside the current approved primitive set

These files are not a stable ontology. They are starter assets for long-horizon
primitive growth.
