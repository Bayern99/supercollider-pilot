# Evaluation Rubric

See also: [docs/design/eval-rubric.md](../../design/eval-rubric.md)

- Was the task executed through the intended Pilot path?
- Did the session produce the required artifact type?
- Is there a usable session summary for later replay?
- Is the review gate explicit when the loop advances a candidate?
- Are repeated failures and preserved items clear enough to feed archive memory?
