# Project Rules

- Keep the control model as `Agent -> Pilot -> SuperCollider`.
- Treat `Pilot` as the main control surface, not as a DAW or plugin host.
- Governed creation loops should prefer workflow and orchestration tools over raw runtime tools.
- Do not encode Zhou Yi meaning, oracle logic, or high-level formation rules directly into low-level SC execution by default.
- Preserve append-only archive history for probe, review, and candidate decisions.
- Avoid scope drift into DAW export, VST hosting, live routing, or mesh multi-agent systems at this stage.
